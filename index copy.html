Desafio Primeiro Repositório GitHub da Dio.

Configurando o seu usuário dentro do Git.

1- Entre no site (https://git-scm.com/) e faça o download.

2- Crie uma pasta em seu computador, dentro dela clique com o botão direito do mouse em Git Bash.

3- Execute esses comando dentro do Git.

4- Git config --global user.name "digite seu nome de usuário"

5- Git config --global user.email "digite o seu email cadastrado no Github"

6- Git config --list

"Seu nome e email aparecerá listado e configurado".
